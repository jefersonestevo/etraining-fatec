Consulta de Usuários

Nesta tela é possível consultar os dados de um usuário através da busca por nome ou matrícula.
*Dica: Se desejar pesquisar todos os alunos do sistema, basta clicar em pesquisar

Campo Nome:
Digite o nome completo do usuário ou parte do nome (Somente letras)

Campo Matrícula:
Digite a matrícula completa do usuário ou parte da matrícula (Somente números)


Tabela de Alunos

Retorna os dados dos alunos conforme o nome ou matrícula informados na pesquisa.
Clique no nome, matrícula, rg ou cpf para editar as informações do usuário.
