Editar Exercício

Nesta tela é possível editar um exercício previamente inserido.

Campos:
Nome: Preencha campo com o nome do exercício
Categoria: Selecione a categoria do exercício
Atividade: Selecione a atividade a ser realizada neste exercício
Pontos por Atividade: Informe a quantidade de pontos que cada realização deste exercício irá gerar para o aluno.

* Todos os campos são obrigatórios.
