Editar Aluno

Nesta tela é possível inserir um novo aluno.

Registro - Campos Obrigatórios:
Nome: Preencha campo com o nome completo do aluno (Somente letras)
Data de Nascimento: Preencha a data de nascimento do aluno no formato dd/mm/aaaa

Matrícula - Campos Obrigatórios:
Dias de Aula: Selecione os dias da semana em que há disponibilidade para o comparecimento às aulas

Dados Corporais - Campos Obrigatórios:
Peso: Preencha o campo com o peso do aluno (Somente números)
Altura: Preencha o campo com a altura do aluno (Somente números)
