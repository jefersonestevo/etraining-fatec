Editar Usuário

Nesta tela é possível editar os dados do usuário, como registro, matrícula e dados corporais.

Registro - Campos Obrigatórios:
Nome: Preencha campo com o nome completo do usuário (Somente letras)
Data de Nascimento: Preencha a data de nascimento do usuário no formato dd/mm/aaaa

Matrícula - Campos Obrigatórios:
Dias de Aula: Selecione os dias da semana em que há disponibilidade para o comparecimento às aulas

Dados Corporais - Campos Obrigatórios:
Peso: Preencha o campo com o peso do usuário (Somente números)
Altura: Preencha o campo com a altura do usuário (Somente números)
